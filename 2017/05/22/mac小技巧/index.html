如何在终端使用指定程序打开文件？

方法一

  1. 打开终端， 输入命令：vi ~/.bash_profile
  2. 在文件末尾添加： alias subl="open -a Sublime\ Text"
  3. 保存后执行 source ~/.bash_profile

缺点： 每次关闭终端后失效，需要在命令行重新运行一次 source ~/.bash_profile 命令。

方法二

  • 命令行用 sublime text
    打开文件

    打开终端， 输入命令 sudo ln /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

    ln 命令： ln 命令是做文件链接用的， 可以理解为建立快捷方式。
    如上命令是将 sublime 提供的 subl 命令链接到 usr/local/bin/subl 这个命令上来。

  • 命令行用 MacDown 打开 .md 文件

    打开终端， 输入命令 open -a macdown test.md
